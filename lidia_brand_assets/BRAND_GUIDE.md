# LidIA brand mini-guide

This pack uses a magenta/red accent for the tribute to your mom while preserving the same waveform-to-notes concept.

## Core colors
- Background start: #2A0E6E
- Background mid: #0F4CCF
- Background end: #10CBEA
- Mark start: #FF4FA3
- Mark mid: #FF5E7A
- Mark end: #7FE7FF

## Use
- Primary website and app branding: `lidia_logo_primary.svg`
- Monochrome/docs/navbar: `lidia_logo_monochrome.svg`
- macOS status bar: `lidia_menu_template.svg`
- Desktop notifications: `lidia_notification_32.png`
- Website favicon: `favicon.ico` and `lidia_favicon.svg`
- macOS app icon: `LidIA.icns` or the `LidIA.iconset/` folder

## UI direction
- Dark navy/slate product UI
- Magenta-to-cyan accent moments only
- Inter for the UI and docs
- Monochrome logo inside dense navigation chrome
- Primary gradient icon for the app, landing page hero, and splash

Headline: Turn conversations into actions.
